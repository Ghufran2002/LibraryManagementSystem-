# 📚 Library Management System

## 📌 Overview

This project is a **Library Management System** developed using Python and SQLite.
It helps manage books, members, and borrowing activities in an efficient and user-friendly way.

---

## 🚀 Features

* 📖 Add, update, and delete books
* 👤 Register and manage members
* 🔄 Issue and return books
* ⚠️ Track overdue books
* 📊 View reports and analytics
* 🗂️ Categorize books

---

## 🛠️ Technologies Used

* Python
* SQLite (Database)
* Tkinter (GUI)

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Ghufran2002/library-management-system.git
```

2. Navigate to project folder:

```
cd library-management-system
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the project:

```
python main.py
```
